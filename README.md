# 3D 太阳系模拟器

一个基于 Three.js 的交互式 3D 太阳系模拟器，使用真实的星球纹理图像，提供沉浸式的太空探索体验。

## 🌟 特性

### 真实星球纹理
- **地球**: 使用高分辨率的卫星图像，展现真实的大陆、海洋和地形
- **月球**: 采用 NASA LRO 任务获得的月球表面图像，显示陨石坑和月海
- **太阳**: 使用太阳表面的真实观测数据，展现太阳的活动细节

### 交互式体验
- **多视角切换**: 宇宙视角、地球视角、月球视角、太阳视角
- **平滑相机动画**: 自然的视角切换动画效果
- **轨道控制**: 鼠标拖拽旋转、滚轮缩放、右键平移

### 物理模拟
- **天体运动**: 地球公转自转、月球绕地球运动
- **比例尺寸**: 相对准确的天体大小和距离比例
- **光照效果**: 太阳光源照明、阴影效果、大气层渲染

### 视觉效果
- **星空背景**: 10,000+ 星点的动态星空
- **星云效果**: 渐变色彩的星云背景
- **大气层**: 地球大气层的半透明效果
- **材质细节**: 凹凸贴图、法线贴图、镜面反射

## 🚀 使用方法

### 在线预览
直接打开 `index.html` 文件即可在浏览器中运行。

### 本地运行
1. 克隆或下载项目文件
2. 确保 `textures/` 文件夹中包含纹理图像
3. 使用本地服务器打开 `index.html`

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

### 控制说明
- **视角切换**: 点击界面左上角的视角按钮
- **动画控制**: 暂停/播放天体运动
- **速度调节**: 拖动滑块调整动画速度
- **鼠标操作**:
  - 左键拖拽: 旋转视角
  - 滚轮: 缩放距离
  - 右键拖拽: 平移视角

## 🛠 技术栈

- **Three.js**: 3D 图形渲染库
- **WebGL**: 硬件加速的 3D 图形
- **Canvas API**: 后备纹理生成
- **Modern JavaScript**: ES6+ 语法和异步操作

## 🎯 核心特性

### 纹理管理系统
- 智能纹理加载：优先使用本地高质量纹理，失败时自动回退到在线资源
- 进度显示：实时显示纹理加载进度和状态
- 后备纹理：程序生成的后备纹理确保始终有可用内容

### 天体物理模拟
- **地球系统**:
  - 23.5° 轴倾斜
  - 自转周期模拟
  - 大气层效果
  - 真实地形凹凸映射

- **月球系统**:
  - 绕地球轨道运动
  - 同步自转（潮汐锁定）
  - 真实表面细节

- **太阳系统**:
  - 中心光源
  - 自转动画
  - 日冕效果

### 视觉增强
- **高级光照**:
  - 点光源（太阳）
  - 环境光
  - 方向光补充
  - 软阴影

- **后处理效果**:
  - 抗锯齿
  - 大气散射
  - 辉光效果

## 🔧 自定义配置

### 修改纹理
替换 `textures/` 文件夹中的图像文件，支持的格式：
- JPG / JPEG
- PNG
- WebP

### 调整天体参数
在 `main.js` 中修改以下参数：
```javascript
// 天体大小
const sunRadius = 8;
const earthRadius = 3;
const moonRadius = 1;

// 轨道距离
const earthOrbitRadius = 60;
const moonOrbitRadius = 10;

// 动画速度
const earthOrbitSpeed = 0.2;
const moonOrbitSpeed = 1;
```

## 🐛 故障排除

### 纹理加载失败
- 检查网络连接
- 确认纹理文件路径正确
- 查看浏览器控制台错误信息

### 性能问题
- 降低浏览器窗口大小
- 关闭其他占用 GPU 的应用
- 更新显卡驱动程序

### 控制不响应
- 检查是否启用了 JavaScript
- 尝试刷新页面
- 检查浏览器兼容性
